# Smart Business System

This is a simple web-based project built to simulate a basic business communication system. It allows users to send messages through a contact form and view them in an admin dashboard.

---

## Features

- Contact form for user messages
- Admin dashboard to view submitted messages
- Ability to delete messages
- Simple and clean UI design
- Local storage used for data saving

---
21
## Technologies Used

- HTML
- CSS
- JavaScript
- LocalStorage (browser storage)

---

## How It Works

1. User fills the contact form
2. Message is saved in browser storage
3. Admin page displays all saved messages
4. Admin can delete messages if needed
